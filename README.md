# Mall Customers Dataset Analysis

This repository contains the dataset **Mall_Customers.csv** and a starter Jupyter notebook for exploratory data analysis (EDA) on mall customer demographics and spending behavior.

## Dataset Description

The dataset includes the following features for each customer:

- **CustomerID**: Unique identifier for the customer  
- **Genre**: Gender of the customer (Male or Female)  
- **Age**: Age of the customer  
- **Annual_Income_(k$)**: Annual income of the customer in thousands of dollars  
- **Spending_Score**: Score assigned by the mall based on customer spending behavior

## Getting Started

1. Clone or download this repository.  
2. Open the `Mall_Customers_Analysis.ipynb` notebook in Jupyter.  
3. Run the cells to perform exploratory data analysis, including:
   - Summary statistics
   - Data visualization (age, income, and spending score distributions)
   - Relationship between income and spending score by gender
